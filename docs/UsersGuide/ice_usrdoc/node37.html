Enabling the Debugger

This section explains how to set some compiler options for debugging. For the coupled model, set DEBUG to TRUE in the env_run.xml script. Before running the model, be sure to delete the object files so that the source code will be recompiled. If a core file is created, it will be in the executable directory. Use dbx to look at the core file. Useful information may also appear in the standard error and output files.
